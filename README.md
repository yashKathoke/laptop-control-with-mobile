# Web-Based Remote Mouse & Keyboard Controller

A simple **remote control app** that allows you to control your laptop's mouse and keyboard using your phone over Wi-Fi. Works via a web browser ‚Äî no installation required on the phone.

---

## Features
- One-finger drag: move mouse
- Tap on touchpad: left click
- Two-finger tap: right click
- Two-finger swipe: scroll
- Keyboard input with full support for letters, numbers, Enter, Backspace, Arrows, and more
- Connect/Disconnect button
